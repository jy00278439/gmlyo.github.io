Python列表类型

发表于 2018-07-10 | 分类于 Python笔记 | 浏览 次

list 列表类型

用途

存放多个值，记录人多个爱好，多个人名

定义方式

在 [] 内用逗号分隔 多个元素 每个元素可以是任意数据类型

list 类型转换的工作原理：list(items)

  1. 先造一个空列表
  2. 类似调用了一个 for 循环，从 items 里取出一个值放入空列表中，循环往复直到取干净为止
    但凡可以被 for循环 循环的，都可以被 list() 转换

调用 list() 方法

常用操作+内置方法

按索引存取值（正向存取+反向存取）：即可以取值，也可以改值

切片（顾头不顾尾，步长）

从一个大列表中切出一个字列表
[起始位置:结束位置:步长]

反向取值，反向步长（顾头不顾尾）

长度 len()

成员运算 in 和 not in

追加 append()

插入 insert()

删除 del, remove(), pop()

del 删除

非列表独有的删除，是一种通用的删除方法

remove() 删除

只是单纯的删除操作，没有返回值

pop() 从列表中取走一个元素值

  1. 删除一个元素
  2. 将该元素当做返回值返回
    默认从最后一个删除

循环（循环取值）

列表可变类型
可变类型：值变，id不变，可变 == 不可hash

reverse() 反转（单纯的反转）

count() 统计列表元素值的个数

clear() 清空列表所有元素值

copy() 列表复制

extend() 向列表末尾追加多个元素值

index()获取列表元素值的索引下标

sort 排序

默认从小到大排序

列表中的元素值，必须是 同一 类型，才能做比较排序

字符串 是 根据位置 比较的，先比较两个字符串的第一个位置的字符
如果 第一个字符比出大小，后面的字符将不再做比较
字符串之间根据 ASCII码 来做比较
小写字母 比 大写字母 大

列表比较大小

队列：先进先出

堆栈：先进后出

总结

  1. 能存多个值，没有类型限制
  2. 有序（能够按索引取值的属于有序的）
  3. 可变类型
    可变：值变，id不变，可变 == 不可hash
    list（任何能被 for循环的对象）转换成 列表
