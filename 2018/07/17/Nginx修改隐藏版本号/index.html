Nginx修改隐藏版本号

Nginx 配置文件

配置文件里增加 server_tokens off;
server_tokens 作用域是 http server location 语句块
server_tokens 默认值是 on，表示显示版本信息，设置 server_tokens 值是 off，就可以在所有地方隐藏 nginx 的版本信息

这样修改之后能隐藏掉 nginx 的版本号，但是还是可以显示 Web 软件为 nginx

注意：

当添加 server_tokens off; 参数的时候，调用的是 src/http/ngx_http_header_filter_module.c 里的值
不添加的时候，显示版本号调用的是 src/core/nginx.h 里的值
所以我们修改的时候需要两个地方都修改。
