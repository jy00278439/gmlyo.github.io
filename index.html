安装Hexo

右击 Git Bash Here
cd /d/Blog/gmlyo/
npm install hexo -g
或
npm install hexo-cli -g
hexo init
或
hexo init blog
cd blog/
npm install
hexo g # 或 hexo generate
hexo s # 或 hexo server，可以在 http://localhost:4000/ 查看
常用 Hexo 命令
生成静态文件，会在当前目录下生成一个新的叫做 public 的文件夹
hexo generate (hexo g)
启动本地 Web 服务，用于博客的预览
hexo server (hexo s)
部署播客到远端（比如github, heroku等平台）
hexo deploy (hexo d)
其他常用命令
hexo new “postName” # 新建文章
hexo new page “pageName” # 新建页面
常用简写
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
常用组合
hexo d -g # 生成部署
hexo s -g # 生成预览
